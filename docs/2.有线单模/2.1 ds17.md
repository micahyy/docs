## 3.快捷键

| 快捷键 | 键值 | 说明 | 备注 |
|--------|------|------|------|
| 长按 NumberLock | LT(1, KC_NUM) | 短按：切换 NumberLock 开关；<br>长按：激活第 1 层功能 | NumberLock 指示灯常亮白色 |
| 长按 NumberLock + 7 | RGB_TOG | 开启 / 关闭 RGB 轴灯 | |
| 长按 NumberLock + 0 | RGB_MOD | 切换 RGB 灯效模式 | |
| 长按 NumberLock + 8 | RGB_SAI | 增加饱和度 | |
| 长按 NumberLock + 2 | RGB_SAD | 减少饱和度 | |
| 长按 NumberLock + 4 | RGB_HUD | 增加色相（色度） | |
| 长按 NumberLock + 6 | RGB_HUI | 减少色相（色度） | |
| 长按 NumberLock + - | RGB_VAI | 增加亮度 | |
| 长按 NumberLock + + | RGB_VAD | 减少亮度 | |
| 长按 NumberLock + * + - | — | 开启 / 关闭全键无冲 | |
| （独立功能） | QK_CLEAR_EEPROM | 在第 7 层的 ENTER 键设为：恢复出厂设置 | |

## 5.固件下载

R3版本固件，不含bootloader，需要使用QMK_TOOLBOX进行刷机


    通过网盘分享的文件：czmao_ds17_v3_3_via.bin
    链接: 
    https://pan.baidu.com/s/1dMgO4e6GqgeQdm2vjf93QA?pwd=38r4 
    提取码: 38r4

R4版本，不含bootloader，需要使用QMK_TOOLBOX进行刷机

R4版本在R3版本上取消了二极管，使用direct pin，每个引脚接地触发方式，与R3版本的性能一致
