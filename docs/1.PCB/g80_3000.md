# 1.1 Cherry G80_3000 改装PCB

## 优联改装PCB

个人对优联改装的看法

1.改装优联是为了它的稳定，省电

2.尽量保证原汁原味

3.在可能的情况下，保证成本最低

所以，在改装的PCB考虑的如下方式

    热插拔+焊接方式：热插拔需要使用定位板，但是原装外壳是无钢设计，又要增加一片定位板；所以 保留两种方式，焊接一次到位，热插拔可以反复折腾


    背光：本身基于K375S制作，K375S不支持灯光，如果要加灯光，需要再额外加主控，耗电又不稳定，所以没有加灯光

最终的方案就是只支持原生的K375S，保证续航和性能



## 有线单模改装PCB

有线单模PCB后期制作，只保留焊接方式，可以使用原装无钢方式无损安装

    支持多配列

    RGB轴灯，下灯位

    键线分离设计，预留原装数据线接口（连接器需要自行购买焊接）
