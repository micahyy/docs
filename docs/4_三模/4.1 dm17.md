## 3.快捷键

| 快捷键              | 键值              | 说明                               | 备注                                   |
| ---------------- | --------------- | -------------------------------- | ------------------------------------ |
| 长按 NumberLock    | LT(1, KC_NUM)   | 短按：切换 NumberLock 开关<br>长按：FN，组合键 | NumberLock 打开时指示灯常亮白色                |
| 长按  *            | LT(2, KC_PSLS)  | 短按：* <br>开关；长按：FN2，组合键触发层2按键     | 长按时 *时 *的指示灯常亮白色                     |
| FN [+] 7         | RGB_TOG         | 开启 / 关闭 RGB 轴灯                   | 按住 NumberLock 再按数字 7                 |
| FN [+] 0         | RGB_MOD         | 切换 RGB 灯效模式                      | 按住 NumberLock 再按数字 0                 |
| FN [+] 8         | RGB_SAI         | 增加饱和度                            | 按住 NumberLock 再按数字 8                 |
| FN [+] 2         | RGB_SAD         | 减少饱和度                            | 按住 NumberLock 再按数字 2                 |
| FN [+] 4         | RGB_HUD         | 增加色相（色度）                         | 按住 NumberLock 再按数字 4                 |
| FN [+] 6         | RGB_HUI         | 减少色相（色度）                         | 按住 NumberLock 再按数字 6                 |
| FN [+] -         | RGB_VAI         | 增加亮度                             | 按住 NumberLock 再按减号                   |
| FN [+] +         | RGB_VAD         | 减少亮度                             | 按住 NumberLock 再按加号                   |
| * [+] -          | NK_TOGG         | 全键无冲/6键无冲切换                      | 按住 NumberLock 再依次按星号和减号 <br>         |
| * [+] NumberLock | QK_CLEAR_EEPROM | 恢复出厂设置                           |                                      |
| * [+] 1          |                 | 切换到蓝牙通道1                         | 长按：指示灯(蓝色)快闪进入配对模式；配对完成后灯光取消蓝色，随灯效变化 |
| * [+] 2          |                 | 切换到蓝牙通道2                         | 长按：指示灯(蓝色)快闪进入配对模式；配对完成后灯光取消蓝色，随灯效变化 |
| * [+] 3          |                 | 切换到蓝牙通道3                         | 长按：指示灯(蓝色)快闪进入配对模式；配对完成后灯光取消蓝色，随灯效变化 |
