# VIA的用法

1.打开网页 via.micah.vip

2.点击设计（齿轮按钮），打开Show Design Tab

![设计齿轮按钮](https://img.micah.vip/i/2026/04/21/69e773dc29f3f.png)

3.点击刷子图标（Design），切换到Design界面，单机Load图标，选择对应的键盘的JSON文件

![](http://img.micah.vip/i/2026/04/21/69e773dc29f3f.png)

4.打开后回到Configure界面，单机Authrie Decive，选择你使用的键盘

![](http://img.micah.vip/i/2026/04/21/69e792b710464.png)

5.确定后即可看到对应的键盘界面，可以根据自己的需求进行改键了

![](http://img.micah.vip/i/2026/04/21/69e792bc6cd5a.png)
