# VIA 使用教程

## 如何连接键盘？

1. 用数据线连接键盘和电脑
2. 打开 https://via.micah.vip
3. 点击右上角 **Settings** → 开启 **Show Design Tab**
4. 返回主界面，点击 **Design** 标签
5. 上传键盘的 JSON 配置文件

## 常见报错

| 报错 | 解决方法 |
|------|----------|
| 找不到设备 | 换一根数据线，或尝试其他 USB 口 |
| 配置文件无效 | 确认 JSON 文件是为你键盘型号准备的 |
