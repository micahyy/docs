# DS22

有线单模数字小键盘

| 列1     | 列2    | 备注               |
| ------ | ----- | ---------------- |
| 按键层数   | 8     | 内容C              |
| 按键数量   | 22    | 内容F              |
| 灯光     | RGB轴灯 | 下灯位              |
| 轴体安装方式 | 轴座热插拔 |                  |
| 键盘接口   | Typec |                  |
| 全键可自定义 | 是     |                  |
| 连接模式   | 有线单模  |                  |
| 蓝牙连接   | 不支持   | 无无线版本            |
| 2.4G连接 | 不支持   | 无无线版本            |
| 扫描率    | 1KHz  |                  |
| 回报率    | 1KHz  |                  |
| 延迟     | 1ms   | 理论延迟，实际可能会在2ms左右 |

# 快捷键

| 组合键           | 键值            | 说明                                | 备注                    |
| ------------- | ------------- | --------------------------------- | --------------------- |
| 长按 NumberLock | LT(1, KC_NUM) | 短按：切换 NumberLock <br>开关；长按：FN，组合键 | NumberLock 打开时指示灯常亮白色 |
| FN+(*)        | MO(2)         | 触发层2按键                            |                       |
| FN+(*)+7      |               | 打开/关闭RGB                          |                       |
| FN+(*)+0      |               | 切换灯效                              |                       |
| FN+(*)+8      |               | 饱和度+                              |                       |
| FN+(*)+2      |               | 饱和度-                              |                       |
| FN+(*)+4      |               | 色度+                               |                       |
| FN+(*)+6      |               | 色度-                               |                       |
| FN+(*)+(-)    |               | 亮度+                               |                       |
| FN+(*)+(+)    |               | 亮度-                               |                       |
| FN+(*)+ent    |               | 打开/关闭全键无冲                         |                       |
| FN+(*)+5      |               | 恢复出厂设置                            |                       |
